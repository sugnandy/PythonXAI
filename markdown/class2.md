## 🧩 一、Streamlit 是什麼？

👉 **Streamlit** 是一個可以把 Python 變成「網頁小程式」的工具
可以：

- 輸入數字
- 按按鈕
- 顯示文字
- 放氣球🎈、下雪❄️

---

## 🔢 二、數字輸入（number_input）

```python
st.number_input("請輸入一個數字")
```

📌 功能：

- 讓使用者輸入數字
- 可以設定最小、最大值

🧠 重點：

- 電腦會記住你輸入的數字
- 之後可以拿來用

---

## 📝 三、成績判斷（if / elif / else）

```python
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
...
```

📌 功能：

- 根據「分數」決定「等第」

🧠 重點：

- **if**：如果
- **elif**：不然如果
- **else**：其他情況
- 程式是「由上往下」判斷的

⚠️ 小提醒
後面出現的不禮貌文字只是「程式練習用」，**實際寫程式不應該這樣用喔** 👍

---

## 🎈 四、按鈕（button）

```python
if st.button("點我"):
    st.balloons()
```

📌 功能：

- 按按鈕會發生事情

🧠 重點：

- 按鈕被按下 → 程式才會做事
- 可以放氣球🎈、下雪❄️

---

## 🔁 五、重複做事情（for 迴圈）

```python
for i in range(5):
    print("hello")
```

📌 功能：

- 重複做一樣的事很多次

🧠 重點：

- `range(5)`：做 5 次（0～4）
- `range(2, 6)`：2 到 5
- `range(2, 10, 2)`：每次加 2

---

## 🔺 六、金字塔（for + 字串）

### 數字金字塔

```python
st.write(str(i) * i)
```

📌 會變成：

```
1
22
333
4444
```

🧠 重點：

- `字串 * 數字` 可以重複顯示

---

### ⭐ 箭頭金字塔

👉 用「空格 + 星星」排出形狀

🧠 重點：

- `" " * 數字` = 空白
- `"*" * 數字` = 星星
- 可以畫圖形！

## 📦 七、串列（List）

```python
L = [1, 2, 3]
```

📌 串列是什麼？

- 像「裝東西的盒子」
- 可以放很多資料

🧠 串列可以放：

- 數字
- 文字
- 混在一起
- 甚至放「另一個串列」

---

### 🔍 取資料

```python
L[0]
```

📌 重點：

- 第一個是 **0**
- 不是 1！

---

## 📏 八、len（長度）

```python
len([1, 2, 3])
```

📌 功能：

- 看串列裡有幾個東西

---

## 🔄 九、走過串列（for）

```python
for element in L:
    print(element)
```

📌 功能：

- 一個一個拿出來用

🧠 比較簡單、常用！

---

## ✏️ 十、改串列內容

```python
a[0] = 10
```

📌 功能：

- 把串列裡的東西換掉

⚠️ 小心！

```python
b = a
```

- a 和 b 會「連在一起」
- 改 b，a 也會變 😱

---

## ➕➖ 十一、串列常用功能

| 功能   | 用法   | 說明           |
| ------ | ------ | -------------- |
| 加東西 | append | 放到最後       |
| 刪指定 | remove | 刪第一個一樣的 |
| 刪最後 | pop    | 拿走最後       |
