## 🐍 Python 基礎語法筆記

### 一、 基礎開發觀念 (class1-1.py)

#### 1. 註解 (Comments)

- **單行註解**：使用 `#`。
- **多行註解**：使用成對的三引號 `"""`。

#### 2. 基本資料型態 (Data Types)

- `int` (整數)：`13`, `10`
- `float` (浮點數)：`3.14`
- `bool` (布林值)：`True`, `False`
- `str` (字串)：`"hello"`

#### 3. 算術運算與字串操作

- **基本運算**：`+`, `-`, `*`, `/`
- **特殊運算**：`//` (整除)、`%` (取餘數)、`**` (次方)。
- **字串操作**：
- `+`：字串連接。
- `*`：字串重複。
- **f-string**：`f"My name is {name}"`，最方便的格式化工具。

#### 4. 資料型態轉換與檢查

- `type()`：檢查資料類型。
- `len()`：檢查字串長度。
- **強制轉型**：`int()`, `float()`, `str()`, `bool()`。
- > **注意**：`bool()` 轉換時，只有 `0`、空字串 `""` 會變成 `False`，其餘通常為 `True`。

#### 5. 使用者輸入 (input)

- `input()` 接收到的資料預設都是 **字串 (str)**，若要進行數學運算，必須先轉型，例如：`int(input())`。

### 二、 Streamlit 套件入門 (class1-2.py)

Streamlit 是 Python 用來快速建立網頁介面的套件。

- `st.title()`：設定網頁大標題。
- `st.write()`：可以輸出多種內容，並支援 **Markdown** 語法：
- `#` 到 `######`：一到六級標題。
- `**粗體**`、`*斜體*`、`[連結名稱](URL)`。
- 使用 ````python` 包裹程式碼區塊。

### 三、 邏輯判斷與流程控制 (class1-3.py)

#### 1. 比較運算子 (Comparison)

會回傳布林值 (`True`/`False`)：

- `==` (等於)、`!=` (不等於)
- `>`、`<`、`>=`、`<=`

#### 2. 邏輯運算子 (Logical)

- `not`：反轉布林值。
- `and`：**兩者皆為真**才為 True。
- `or`：**其中之一為真**即為 True。

#### 3. 條件判斷 (If-elif-else)

這是程式的「大腦」，決定執行的路徑：

```python
if 條件1:
    # 條件1成立執行
elif 條件2:
    # 條件1不成立，但條件2成立執行
else:
    # 以上都不成立才執行

```
