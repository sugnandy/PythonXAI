## 🍔 class3-1：點餐機（Streamlit）

### 這一段在做什麼？

做一個 **可以點餐、加入購物籃、刪除餐點的小網頁**。

### 重要觀念

- **Streamlit**：可以用 Python 做網頁
- **session_state（ss）**：像一個「記憶盒子」，幫我們記住點了什麼
- **list（清單）**：用來存很多餐點

### 重點功能

- 輸入餐點 ➜ 按「加入」 ➜ 餐點進購物籃
- 每個餐點旁邊都有「刪除」按鈕
- `st.rerun()`：畫面重新整理

📌 想像成：

> 你在自助點餐機點漢堡🍔，點了就放進購物籃🧺

## ➕ class3-2：算術指定運算子 + 迴圈

### ✏️ 算術指定運算子（偷懶寫法）

```python
a += b  # a = a + b
a -= b  # a = a - b
a *= b  # a = a * b
a /= b  # a = a / b
```

📌 就像「快速計算鍵」，比較省力！

---

### 🔁 while 迴圈（一直做，直到不符合條件）

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

➡️ 印出 0～4

### ⛔ break（強制停止）

- 遇到 `break` 就立刻停下來

---

### 🔁 for 迴圈

```python
for i in range(1, 6):
    print(i)
```

➡️ 印出 1～5

📌 迴圈就像「一直做一樣的事」

## 🎲 class3-3：random 隨機數

### 🎯 random 在做什麼？

讓電腦 **隨機出數字**，像擲骰子🎲

### 常用指令

- `randrange(1, 6)` ➜ 1～5
- `randint(1, 5)` ➜ 1～5（包含 5）

### 統計出現次數

- 用 **list（清單）** 來記錄每個數字出現幾次

📌 用來做：

> 抽獎、遊戲、猜數字 🎮

---

## 🔢 class3-4：猜數字（文字版）

### 遊戲規則

- 電腦先想一個 **1～100 的數字**
- 玩家一直猜
- 電腦提示：
  - 太小 ➜「再大一點」
  - 太大 ➜「再小一點」
  - 剛好 ➜「猜對了！」

### 用到的重點

- `while True`：一直玩到猜對
- `break`：猜對就結束

## 🖥️ class3-5：猜數字（網頁版）

### 跟上一版差在哪？

✔️ 變成 **網頁遊戲**
✔️ 用按鈕和輸入框
✔️ 猜對會 🎈 放氣球

## 📦 class3-6：字典（dictionary）

### 🗂️ 字典是什麼？

👉 一種「**配對表**」

```python
{"A": 90, "B": 80}
```

📌 像成績表：

- A ➜ 90 分
- B ➜ 80 分

---

### 常用功能

- 取資料：`D["A"]`
- 新增：`D["f"] = 6`
- 修改：`D["a"] = 10`
- 刪除：`pop("c")`

---

### 🔍 檢查有沒有

```python
"d" in D3   # True
```

---

### 📊 實際應用：成績平均

- 每一科是一個 key
- 分數用 list 存
- 用迴圈算平均

📌 很像老師在算全班平均分數 👩‍🏫
